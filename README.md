# Group 12 – Data Fitting and OLS Methods

## Mô tả đồ án

Đồ án nghiên cứu về **Data Fitting** và **Ordinary Least Squares (OLS)** methods, bao gồm:

- **Part 1**: Lý thuyết và cài đặt OLS từ đầu (from scratch), Ridge/Lasso regression, phân tích phần dư, và cross-validation.
- **Part 2**: Ứng dụng thực tế trên bộ dữ liệu thực, tiền xử lý dữ liệu, so sánh mô hình, và các phương pháp nâng cao (Kernel/Bayesian regression).
